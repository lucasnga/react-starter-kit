# React starter kit

Simple app starter with react, bootstrap and pug.

## Using

- es6
- bootstrap
- sass
- pug
- react
- webpack
- webpack-dev-server

## Running

What's needed:

[node](https://nodejs.org/en/)

[yarn](https://github.com/yarnpkg/yarn)

- development

`yarn start`

- production

`yarn build`
